# 📝 iNotebook - Secure Cloud Notes App

iNotebook is a **full-stack MERN (MongoDB, Express, React, Node.js) application** that allows users to securely store and manage their personal notes in the cloud.

The application provides **authentication, note management, and secure data storage**, allowing users to create, read, update, and delete notes with a smooth and responsive user experience.

---

## 🚀 Features

- 🔐 Secure User Authentication (Signup / Login)
- 📝 Create Notes
- 📖 View Notes
- ✏️ Edit Notes
- ❌ Delete Notes
- ☁️ Cloud Storage using MongoDB
- ⚡ Fast and Responsive React Frontend
- 🛡 JWT Based Authentication

---
